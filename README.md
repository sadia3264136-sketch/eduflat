# EduFlow — Flat Structure (Easiest to Upload)

Every file is in the ROOT — no `src` or `app` folder to mess up.

## Deploy Steps

1. Extract this zip
2. GitHub → New Repository → name `eduflow`
3. Upload ALL files you see in this folder directly (select all, drag & drop)
4. On Vercel: Add New Project → select repo → Framework: **Vite** → Deploy
